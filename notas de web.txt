¿Es la web y el internet lo mismo?  
WWW son las siglas World Wide Web  que es un Sistema de archivos que se encuentra interconectado entre si que se accede mediante internet. Todo portal o pagina  web que se encuentra en internet posee el prefijo WWW para hacer referencia conjunto de datos (textos, archivos, imágenes y datos) que se encuentran agrupados en los mencionados portales.

Lenguajes de programación web
CSS (CASCADING STYLE SHEETS) Hojas de Estilo en cascada – Que es un lenguaje que describe la presentación de los documentos estructurados en hojas de estilo para diferentes métodos de interpretación, es decir, describe como se va a mostrar un documento en pantalla, por impresora.
JS Es un lenguaje de programación que te permite crear contenido nuevo y dinámico, controlar archivos multimedia, crear imágenes animadas y muchas otras cosas más. 
PHP  (acrónimo recursivo de PHP: Hypertext Preprocessor) es un lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web y que puede ser incrustado en HTML.
ASP.NET Es un entorno para aplicaciones web desarrollado y comercializado por Microsoft.Es usado por programadores y diseñadores para construir sitios web dinámicos, aplicaciones web y servicios web XML.

HTML Hypertext Markup Language (Lenguaje de marcas de Hipertexto) hace referencia al lenguaje de marcado para la elaboración de paginas web.
Es un lenguaje de marcado que se utiliza para el desarrollo de paginas de internet. Se trata de la sigla que corresponde a Hypertext markup Language, es decir, Lenguaje de Marcas de Hipertexto, que podría ser tarducido como lenguaje de formato de documentos para hipertexto.

Arquitectura de componentes y conectores
Cliente/servidor
Modelo computacional
•	Una serie de componentes (servidores) proporcionan servicios y aguardan a que otros componentes (clientes) lo soliciten.
•	Aplicaciones: 
o	Servidores de nombre
o	Servidores de correo
o	Servidores web
Compoenentes/conectores
	Componentes
		Conformado por clientes y servidores.
•	Los servidores hacen pública una colección de operaciones/servicios.
•	Varios servidores y distintas topologías.
Conectores 
	A través de intercambios peticiones o solicitudes y respuestas.
•	Interacción punto a punto: Asimétrica: iniciada por los clientes vs procesos reactivos: el cliente debe conocer la identidad del servidor.
Según la forma de atender el servicio, se puede hacer lo siguiente
	Clasificación de los servidores:
•	Servidores iterativos:
Servidor que además de recoger la petición de servicio se encarga de atender el mismo
•	Servidores concurrentes:
Servidor que recoge cada una de las peticiones de los clientes y crea procesos para que se encarguen de atenderlos. Esto especialmente útil en los servidores que han de 
atender un gran volumen de peticiones.

HTML
HYPERTEXT MACKUP Language, es un sistema estandarizado para etiquetar archivos de texto para crear páginas web con diferentes tipos de fuentes, colores gráficos y/o efectos. Por lo tanto está compuesto de un conjunto de etiquetas.
Estas etiquetas HTML son palabras clave y atributos rodeados por los signos mayor y menor.
Las etiquetas de html se emplean para el estructurado semántico del contenido.

XML
Es un lenguaje que permite la organización y el etiquetado de documentos. Esto quiere decir que el XML no es un lenguaje en sí mismo, sino un sistema que permite definir Lenguajes de acuerdo a las necesidades. El XHTML el MathML y el SVG son algunos lenguajes que el XML tiene la capacidad de definir.
